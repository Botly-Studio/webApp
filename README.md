## Botly Studio - WebApp

### Stable version

#### Local Install

```bash
git clone git@gitlab.com:lamachinerie/botly/botlystudio/webapp.git
cd webapp
npm install
```

#### Docker install

```bash
git clone git@gitlab.com:lamachinerie/botly/botlystudio/webapp.git
cd webapp
docker build -t botlystudio-webapp .
```
